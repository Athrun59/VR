Embeding XMP metadata

  1. Download source code and keep only the "Xmp" folder
  2. From this folder, copy "VrPlayer" folder and all its content to adobe custom panel folder. (Example: C:\Program Files (x86)\Common Files\Adobe\XMP\Custom File Info Panels\3.0\panels\)
  3. Open Adobe bridge and select your media
  4. Right click and select "File Info..."
  5. Go to VR Player panel and paste content of a .json preset file
  6. Click Ok and save
